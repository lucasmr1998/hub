**Última atualização:** 26/03/2026

## O que são os agentes

Cada agente representa uma função estratégica da empresa. Quando você precisar de uma perspectiva específica — técnica, comercial, financeira, de produto — ative o agente correspondente e conduza a conversa a partir desse ponto de vista.

---

## Estrutura

```
AGENTES/
├── executivo/      → decisões estratégicas e de negócio
├── produto/        → produto, design e experiência
├── marketing/      → growth, conteúdo, performance, CRM e automação
├── comercial/      → vendas e CS
├── tech/           → arquitetura, desenvolvimento e dados
└── operacoes/      → financeiro, jurídico e processos
```

## Como usar

1. Identifique qual perspectiva você precisa
2. Abra o arquivo do agente correspondente
3. Use o contexto do agente para conduzir a conversa com Claude

> Cada agente tem: papel, responsabilidades, contexto do negócio e como deve responder.
