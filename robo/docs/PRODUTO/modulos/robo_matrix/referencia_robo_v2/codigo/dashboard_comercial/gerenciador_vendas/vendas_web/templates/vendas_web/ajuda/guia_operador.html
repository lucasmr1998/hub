Guia do Operador

Você conduz o pipeline de Indicação (e, conforme seu perfil, o de Atendimento) do começo ao fim, na mão — é o único fluxo do CRM que não anda sozinho.

O que é uma Indicação

É um lead trazido por alguém — um cliente que indicou um conhecido, uma campanha externa, etc. Diferente da Aquisição (que vem do robô no WhatsApp), a Indicação nasce de um cadastro manual feito por você, e carrega um código do indicador para rastreio (quem trouxe aquele cliente).

A jornada, passo a passo

1. Cadastrar 2. Completar dados 3. Converter em cliente 4. Contrato aceito 5. Abrir atendimento + O.S.

1. Cadastrar a indicação

No Pipeline, na aba Indicação, use o botão + Nova Indicação. Preencha nome, telefone e o código do indicador — o resto pode ser completado depois. Ao salvar, a oportunidade já aparece no primeiro estágio do funil.

2. Completar os dados

Abra a oportunidade e use o botão Ações do Operador — ele abre um painel central com um formulário para preencher CPF, endereço, plano e demais dados obrigatórios. Salve antes de seguir para a conversão.

3. Converter em cliente

Com os dados completos, clique em Converter em cliente. Nos bastidores, em segundos, o sistema faz duas chamadas ao HubSoft:

  1. Cria o prospecto (cadastro inicial no HubSoft).
  2. Converte o prospecto em cliente, já com o plano escolhido — via API própria, bem mais rápida que o processo manual antigo.
Se faltar algum dado obrigatório, o botão avisa exatamente o que falta — não precisa adivinhar.

4. Acompanhar o contrato

Depois da conversão, o HubSoft gera o contrato do cliente. Você não aceita por ele — o próprio cliente recebe e assina pelo app da Megalink. Sua parte é acompanhar: o painel mostra um selo Aguardando aceite (laranja) ou Aceito (verde), e o botão Atualizar status consulta o HubSoft na hora.

O botão de abrir atendimento fica bloqueado enquanto o contrato não estiver aceito — é uma trava do sistema, não um bug. Assim que o cliente aceitar no app, o selo vira verde e o botão libera sozinho.

5. Abrir atendimento + O.S.

Com o contrato aceito, escolha turno e data e clique em Abrir atendimento + O.S.. O sistema agenda automaticamente no Matrix — abre o atendimento e a ordem de serviço de instalação já com a data escolhida. A oportunidade avança para o estágio final do funil.

Movendo o card manualmente

Além dos botões de ação, você também pode arrastar o card entre as colunas do kanban a qualquer momento — útil para marcar como Perdido se a indicação não avançar, por exemplo. Toda mudança de estágio fica registrada no histórico da oportunidade.

Falar com o lead

O botão verde de WhatsApp na tela da oportunidade já abre com uma mensagem inicial pronta (configurada pelo Administrador para o pipeline de Indicação) — é só conferir e enviar.

Notificações

Você é avisado automaticamente quando uma nova indicação entra no funil, quando uma conversão em cliente é concluída e quando o atendimento/O.S. é aberto com sucesso. Se uma conversão ou um agendamento falhar, quem é avisado é o gerente/administrador — a mensagem de erro na tela já te diz o que aconteceu, para tentar de novo. Acompanhe tudo pelo sino da topbar.
