Guia do Auditor

Seu perfil é somente leitura: você enxerga tudo, em todos os pipelines, mas não altera nada. Ideal para revisão, conferência e compliance.

O que você vê

Diferente dos demais perfis, o seu escopo de dados é sempre todos os registros — você enxerga as oportunidades de todos os vendedores e operadores, em qualquer pipeline liberado para o seu perfil, sem o filtro "só as minhas".

  • Todas as abas de pipeline que o Administrador liberar para o seu perfil.
  • O histórico completo de mudanças de estágio de cada oportunidade (aba Hist. Estágios), com data, hora e quem moveu.
  • Notas internas, tarefas e o status do contrato de cada cliente.
  • Relatórios de Desempenho e Retenção, se liberados para o seu perfil.

O que você não faz

Nenhum botão de ação funciona para o seu perfil por padrão: mover card no kanban, editar dados de um lead, converter uma indicação em cliente, abrir atendimento/O.S., atribuir responsável. Se você tentar mesmo assim (inclusive digitando um link direto), o sistema bloqueia a ação com um erro de permissão — é esperado, não é falha do sistema.

Se você precisar registrar uma observação sobre algo que auditou, peça a um Gerente ou Operador para adicionar uma nota na oportunidade — ou combine com o Administrador uma capacidade extra específica para o seu caso.

Conferindo contratos

Dentro de uma oportunidade convertida em cliente, o painel mostra o status real do contrato — Aceito (com data) ou Aguardando aceite — direto do HubSoft. Útil para conferir se nenhuma instalação foi agendada antes do aceite (o sistema já impede isso na origem, mas a auditoria confirma).

Onde encontrar o histórico

  • Timeline da oportunidade — todos os contatos e eventos, em ordem cronológica.
  • Hist. Estágios — cada transição de coluna no kanban, com responsável e tempo gasto no estágio anterior.
  • Admin Django (se liberado) — registro técnico de cada execução de automação no HubSoft, com sucesso/falha e mensagem de erro, para investigação mais profunda.
