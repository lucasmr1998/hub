Como funciona uma interação
O caminho que cada mensagem do cliente percorre
1
Cliente envia msg
WhatsApp → Matrix recebe mensagem do cliente.
2
/proximo-passo
Matrix consulta API IA. Decide se mostra menu, retoma fluxo, transborda ou encerra.
3
Bot pergunta
Matrix exibe a mensagem retornada (msg_pergunta). Cliente responde.
4
/validar
API IA aplica regra (extractor), executa hooks especiais, salva no lead.
5
Resposta
Se válido → loop volta. Se inválido → pede de novo. Se transbordo → atendente.
