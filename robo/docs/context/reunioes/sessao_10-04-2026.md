## Bug Pendente (proximo sessao)

**Extracao IA falhando intermitentemente no fluxo FATEPI:**
- A API OpenAI funciona (teste direto retorna JSON correto)
- Mas no engine, a extracao as vezes retorna sucesso_extracao=False
- Causa provavel: o JSON da resposta nao esta sendo parseado corretamente em alguns casos, ou a variavel `_ultima_mensagem` nao esta sendo atualizada corretamente entre nodos
- Resultado: respostas validas (nome, curso, ingresso) caem no fallback do agente IA
- Investigar: adicionar logs detalhados na funcao `_processar_ia_questao` para ver exatamente o que a IA retorna e como o engine processa
