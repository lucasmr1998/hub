## Decisões tomadas

| Decisão | Motivo |
|---------|--------|
| 3 tiers: Starter / Start / Pro | Starter para ISPs pequenos com barreira de entrada mínima |
| Escada 1:2:3,5 | Salto para Pro menor do que dobrar, facilita conversão |
| Comercial: R$397 / R$797 / R$1.397 | Produto âncora, maior valor percebido |
| Marketing: R$297 / R$597 / R$997 | Ligeiramente abaixo do Comercial |
| CS: R$247 / R$497 / R$897 | Mais acessível, clube tem custo para o ISP manter |
| Transacional Comercial: R$7/venda | Alinha receita Aurora com conversão do provedor |
| Transacional Marketing: R$0,05/contato/mês | Por base acumulada, escala com crescimento |
| Transacional CS: R$0,15/cliente ativo no clube | Mercado pratica R$0,10 a R$0,20 |
| Trial 14 dias só para HubSoft | ICP perfeito, integração nativa, resultado rápido |
| Comissão do sócio: 30% do lucro líquido | Modelo societário já definido |
| Demo ao vivo | Parceiro conduz, Aurora apoia nas primeiras |
| Réguas padrão em `docs/PRODUTO/` | São especificação de produto, não estratégia GTM |
| Fase 0 encerra 30/03, refatoração até 27/03 | Multi-tenancy crítico para virar SaaS |
| Meta Fase 1: 15 clientes até junho/2026 | Primeiro trimestre de operação |
