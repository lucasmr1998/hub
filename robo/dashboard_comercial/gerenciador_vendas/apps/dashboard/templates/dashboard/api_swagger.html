🚀 API Dashboard Comercial

Documentação interativa e completa da API

📋 APIs Públicas (Registro/Atualização)

Endpoints para criar e atualizar dados. Não requerem autenticação.

  • POST /api/leads/registrar/
  • POST /api/leads/atualizar/
  • POST /api/prospectos/registrar/
  • POST /api/prospectos/atualizar/
  • POST /api/historicos/registrar/
  • POST /api/historicos/atualizar/

🔍 APIs de Consulta

Endpoints para consultar dados. Requerem autenticação.

  • GET /api/consultar/leads/
  • GET /api/consultar/historicos/
  • GET /api/consultar/fluxos/
  • GET /api/consultar/questoes/
  • GET /api/consultar/atendimentos/
  • GET /api/consultar/respostas/

📊 APIs do Dashboard

Endpoints para dados do dashboard. Requerem autenticação.

  • GET /api/dashboard/data/
  • GET /api/dashboard/charts/
  • GET /api/dashboard/prospectos/

💰 APIs de Vendas

Endpoints para validação de vendas. Requerem autenticação.

  • POST /api/vendas/aprovar/
  • POST /api/vendas/rejeitar/
  • GET /api/vendas/historico-contatos/

🤖 APIs N8N

Endpoints específicos para integração com N8N. Permitem controle completo dos fluxos de atendimento.

  • GET /api/n8n/lead/buscar/
  • POST /api/n8n/lead/criar/
  • GET /api/n8n/fluxos/
  • POST /api/n8n/atendimento/iniciar/
  • GET /api/n8n/atendimento/{id}/consultar/
  • POST /api/n8n/atendimento/{id}/responder/
  • POST /api/n8n/atendimento/{id}/avancar/
  • POST /api/n8n/atendimento/{id}/finalizar/

🔧 Características Especiais

Recursos avançados do sistema:

  • ✅ Status configuráveis via admin
  • ✅ Logs automáticos de todas operações
  • ✅ Relacionamentos automáticos via id_hubsoft
  • ✅ Suporte a múltiplos formatos de data
  • ✅ Validação de dados robusta
  • ✅ Integração completa com N8N
  • ✅ Contabilização automática de questões
  • ✅ Fluxos de atendimento dinâmicos
