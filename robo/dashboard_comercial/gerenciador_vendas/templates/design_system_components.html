{# ========== LIST FILTERS ========== #}

List Filters

Barra de filtros pra listagens de entidades. Dois modos: compact (tabs + search) ou grid (multiplos campos).

Modo A — compact (tabs + search em linha)

Uso ideal: listagens com poucos estados (status, categoria). Tabs na esquerda, search na direita.

{% include "components/list_filters.html" with tabs=filter_tabs_demo search_placeholder="Buscar template..." search_id="demoSearchA" search_oninput="console.log(this.value)" %}
{% verbatim %}{% include "components/list_filters.html" with tabs=status_tabs search_placeholder="Buscar..." search_oninput="fn(this.value)" %}{% endverbatim %} Modo B — grid (multiplos campos)

Uso ideal: listagens grandes com varios criterios (leads, oportunidades, tickets). Campos em grid responsivo + busca textual embaixo.

{% include "components/list_filters.html" with fields=filter_fields_demo search_placeholder="Buscar por nome ou telefone..." clear_url="#" %}
{% verbatim %}{% include "components/list_filters.html" with fields=filter_fields search_placeholder="Buscar..." clear_url=request.path %}{% endverbatim %} Modo B — colapsado (preserva espaco)

Passar collapsed=True faz o filtro comecar fechado. Clica no header pra expandir. Ideal pra kanban, listas grandes onde o conteudo e a estrela. Opcional: active_filters_count mostra badge com numero de filtros ativos.
