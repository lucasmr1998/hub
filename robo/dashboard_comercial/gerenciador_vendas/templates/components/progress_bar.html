{% load l10n %} {% comment %} Progress bar — barra fina de progresso, pra usar dentro de celula de tabela, card ou linha de lista. Versao enxuta do breakdown_row (que carrega label, meta e bloco de stats e nao cabe numa celula). Params: percent: numero 0-100 (largura da barra) variant: primary (default) | success | danger | warning | info — cor da barra label: texto opcional acima da barra (ex: "3/7 respondidos") width: largura maxima em px (default: 160) Uso: {% include "components/progress_bar.html" with percent=42 label="3/7" %} Sobre o `unlocalize`: o projeto roda em pt-br com USE_L10N, entao o Django renderiza 50.0 como "50,0". Dentro de style="width: ...%" isso vira CSS invalido, o navegador descarta a regra e TODA barra aparece cheia. O bug e silencioso: nao quebra a pagina, so mostra numero errado. Percentual quase sempre e float, entao sem unlocalize o componente so funciona por acaso quando o valor e inteiro. {% endcomment %}
